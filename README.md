# Color Badge Menu

A simple HTML and CSS project that demonstrates the use of colors to communicate meaning in a user interface. The project uses colored status badges to indicate product availability while practicing text colors, background colors, spacing, and layout design.

## Features

- Centered card layout
- Color-coded status badges
- Visual hierarchy using text colors
- Rounded badge design
- Clean and simple user interface
- Responsive card styling

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
color-badge-menu/
├── index.html
├── styles.css
└── README.md
```

## Concepts Practiced

### Background Colors
Used different background colors to separate content and improve visual design.

### Text Colors
Applied color hierarchy to distinguish headings, subtitles, and status information.

### UI Status Indicators
Used color-coded badges to communicate availability:

- Green = In Stock
- Yellow = Low Stock
- Red = Sold Out

### Layout and Spacing
Used padding, borders, and border-radius to create a clean card layout.

### Flexbox
Used Flexbox to align product names and badges efficiently.
